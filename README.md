# BirBak Nedir?


BirBak Flutter Turkey Hackhaton kapsamında gerçek ihtiyaç sahiplerinin hızlı bir şekilde mağduriyetlerini gidermek için üretilen bir uygulama projesidir.

## Projenin Kapsamı:
